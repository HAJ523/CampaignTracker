About

The primary use for the campaign tracker was to satisfy my own personal needs. Despite my use being mainly limited to a specific pen and paper RPG I wanted to make sure that I generalized where possible. There were a few core ideas that I felt were missing from other trackers and so this project was born. wiki2html This API is used to translate the journal text into html. It has been modified from it's original form to accomidate the added functionality of the journal such as rolling and image display.

Leaflet This API is used create the custom map pages which allow for markers and the measuring of distance. In addition to this API the measureControl plugin is also used.

Fantasy Name Generators Much of the raw data for the name generation comes directly from this resource. Ported into this application to allow for access during campaigns without internet access.
